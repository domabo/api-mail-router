=== API Mail Router === 
Contributors: Domabo 
Tags: MailChimp, Mandrill, Facebook, API, Webhook, Asana
Stable tag: 0.1
Requires at least: 3.0
Tested up to: 3.6
 
This is a plugin that processes web hooks from FaceBook and MailChimp and posts to MailChimp, Mandrill, and Asana

== Installation ==
1. Download the .zip file
2. Unpack into wp-content/plugins/ folder
3. Activate the plugin
